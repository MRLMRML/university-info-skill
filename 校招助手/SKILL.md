# 校招助手 Skill

## 概述

面向 HR 和校招团队的决策支持工具。基于高校学科实力、毕业生流向、行业匹配度等数据，帮助制定校园招聘策略。

本 Skill 与「高校信息检索」Skill 配合使用 — 本 Skill 专注校招决策场景，高校基础信息查询请使用高校信息检索 Skill。

## 数据文件

| 文件 | 内容 | 用途 |
|------|------|------|
| `data/recruitment.json` | 行业→目标校映射、校招策略模板、ROI估算模型 | 目标校推荐、策略生成 |
| `data/talent_flow.csv` | 主要985高校毕业生城市/行业流向数据 | 流向分析 |

同时可读取「高校信息检索」Skill 的数据文件获取学科评估、排名等信息。

## 核心能力

### 1. 目标校筛选

**触发词**：目标校、去哪些学校招人、校招目标

**处理逻辑**：
1. 识别用户的行业/岗位需求
2. 从 recruitment.json 中匹配行业→目标校映射
3. 从高校信息检索 Skill 的 disciplines.csv 获取学科评级细节
4. 从 rankings.csv 获取排名权重
5. 按核心校/重点校/储备校三梯队输出

**输出模板**：
```
🎯 校招目标校推荐 — {行业/岗位}

📋 核心校（相关学科 A+/A，排名 TOP20）
1. {校名} — {学科评级} | 软科#{排名} | {匹配理由}
2. ...

📋 重点校（相关学科 A-/B+，排名 TOP50）
1. {校名} — {学科评级} | 软科#{排名} | {匹配理由}
2. ...

📋 储备校（性价比优，竞争压力小）
1. {校名} — {学科评级} | {匹配理由}
2. ...

💡 建议：核心校重点投入宣讲+提前批，储备校可线上宣讲降低成本
```

### 2. 校招策略报告生成

**触发词**：校招策略、校招计划、校招方案、招聘规划

**处理逻辑**：
1. 收集需求：行业、岗位、人数、预算、地域偏好
2. 按 recruitment.json 中的报告模板结构生成
3. 整合目标校数据、ROI估算、排期建议

**输出模板**（按 recruitment.json 中的"校招策略模板.报告结构"输出完整六章报告）

### 3. 招聘 ROI 估算

**触发词**：校招ROI、招聘预算、成本估算、投入产出

**处理逻辑**：
1. 基于 recruitment.json 中的 ROI 估算模型
2. 结合目标校层次（985/211）对应的转化率参考值
3. 计算预估的宣讲→投递→面试→入职漏斗

**输出模板**：
```
💰 校招 ROI 估算 — {公司/岗位}

📊 招聘漏斗预估（按 {N} 所目标校）
| 阶段 | 985院校 | 211院校 |
|------|---------|---------|
| 宣讲参与 | ~{N}人/场 | ~{N}人/场 |
| 简历投递 | ~{N}% | ~{N}% |
| 面试通过 | ~{N}% | ~{N}% |
| Offer接受 | ~{N}% | ~{N}% |

💵 成本估算
• 单场宣讲成本：约 {N} 元
• {M} 场总成本：约 {N} 元
• 预估入职人数：{N} 人
• 人均招聘成本：约 {N} 元/人

💡 优化建议：{降本增效建议}
```

### 4. 毕业生流向分析

**触发词**：毕业生去向、人才流向、去了哪些公司、哪些城市

**数据源**：`data/talent_flow.csv` + 高校信息检索 Skill 的就业概况

**输出模板**：
```
🔄 {高校名称} 毕业生流向分析

🏙️ 城市流向 TOP5
1. {城市} — {占比}%
2. ...

🏢 行业流向 TOP5
1. {行业} — {占比}%
2. ...

🏭 代表性去向企业
• {企业列表}

📊 数据来源：{高校名}就业质量报告（{年份}）
```

### 5. 竞品校招情报

**触发词**：竞品校招、竞争对手去了哪些学校、行业校招动态

**处理逻辑**：通过 web search 搜索 "{竞品公司} 校招 {年份} 目标校" 获取信息，结合本地行业→目标校映射分析

## 注意事项

- ROI 数据为行业参考值，实际以企业历史数据为准
- 流向数据基于各校公开就业报告，存在滞后性
- 竞品情报依赖线上搜索，信息时效性和完整性有限
