# 贡献指南

感谢你对本项目的关注！欢迎通过以下方式贡献：

## 数据修正

如果发现数据错误，请提交 Issue 或 PR：

1. Fork 本仓库
2. 修改对应数据文件
3. 在 PR 描述中附上**数据来源链接**（必须为官方/权威来源）
4. 运行验证脚本确认无误：`python scripts/validate_data.py`

### 常见修正类型

| 类型 | 文件 | 要求 |
|------|------|------|
| 学科评估结果 | `高校信息检索/data/disciplines.csv` | 附教育部学位中心来源 |
| 高校基础信息 | `高校信息检索/data/universities.json` | 附各校官网或教育部链接 |
| 就业/科研数据 | `高校信息检索/data/universities.json` | 附就业质量报告链接 |
| 排名数据 | `高校信息检索/data/rankings.csv` | 附软科官网链接 |

## 数据补充

欢迎补充以下数据：

- 更多高校的就业/科研/师资/生源数据
- 录取分数线数据（scores.db）
- 毕业生流向数据（talent_flow.csv）
- 考研报录比数据（admission.db）

## 功能改进

如果有新的查询场景或功能建议：

1. 先提 Issue 讨论
2. 确认方向后再提 PR
3. 修改 SKILL.md 时确保输出模板格式一致

## 提交规范

Commit message 格式：

```
<type>(<scope>): <description>

feat: 新功能
fix: 数据修正
data: 数据更新（不改功能）
docs: 文档更新
```

示例：
- `fix: 修正中山大学一流学科列表`
- `data: 补充2024年各省录取分数线`
- `feat(2.7): 新增专业目录查询`

## Issue 模板

### 数据错误报告

```
**错误位置**: [文件名 + 具体字段]
**当前数据**: [错误内容]
**正确数据**: [正确内容]
**来源链接**: [权威来源URL]
```

### 功能建议

```
**使用场景**: [描述你的使用场景]
**期望行为**: [你希望 Skill 怎么做]
**参考数据源**: [如果需要新数据，数据从哪获取]
```
