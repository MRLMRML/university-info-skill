# 高校信息检索 Skill

## 概述

面向校园招聘、校企合作、学术研究、综合评估等场景的中国高校信息查询工具。本地数据覆盖全国主要高校的基础信息和学科评估结果，不足时通过线上搜索补充。

## 数据文件

本 Skill 附带以下本地数据（相对于本文件所在目录）：

| 文件 | 内容 | 可信度 | 用途 |
|------|------|--------|------|
| `data/universities.json` | ~100所高校的层次、城市、类型、一流学科、官网、就业概况、科研概况 | 层次/城市/类型=权威；就业/科研=近似估算 | 单校查询的主要数据源 |
| `data/disciplines.csv` | 第四轮学科评估全部95个学科的A+院校（210条），源自教育部2017年公布结果 | 权威（教育部原始数据） | 学科实力查询 |
| `data/config.yaml` | 985/211/双一流名单、高校类型分类、地区分布、检索配置 | 权威（教育部公开文件） | 筛选和分类查询 |

**数据可信度说明**：
- 「权威」= 直接源自教育部公开文件，可作为事实引用
- 「近似估算」= 基于各校公开报告的概括性数据，仅供参考，具体数字可能与最新官方发布有偏差
- 就业/科研/生源数据建议在输出时标注"数据为概括性参考，具体请查阅各校最新官方报告"

**使用顺序**：先查本地数据 → 本地数据不足时，使用 web search 工具补充 → 整合后输出。

## 查询类型与处理策略

### 1. 基础信息查询

**触发词**：基本信息、简介、什么层次、是不是985/211、在哪个城市

**数据源**：`universities.json` → 层次、城市、类型、网址

**输出模板**：
```
📚 {高校名称}

🏛️ 基础信息
• 院校层次：{985|211|双一流}
• 办学性质：{公立/民办}
• 所在城市：{城市}
• 高校类型：{综合/理工/师范/...}
• 官方网站：{网址}
```

### 2. 学科实力查询

**触发词**：学科评估、A+学科、优势学科、什么学科强、学科排名

**数据源**：`disciplines.csv`（按院校/学科名筛选）+ `universities.json`（一流学科字段）

**处理逻辑**：
1. 若问"XX大学有哪些强势学科" → 从 disciplines.csv 筛选该校所有A类学科，按评估结果(A+ > A > A-)排序
2. 若问"XX学科哪些学校强" → 从 disciplines.csv 筛选该学科所有院校，按评估结果排序
3. 一流学科信息从 universities.json 补充

**输出模板**：
```
📖 {高校名称} - 学科实力

🏆 A+学科（{N}个）
• {学科名} - 学科代码 {代码}
...

🥇 A学科（{N}个）
• {学科名}
...

🥈 A-学科（{N}个）
• {学科名}
...

🌟 双一流建设学科
• {一流学科列表}

📊 数据来源：教育部第四轮学科评估（2017年公布）
```

### 3. 科研水平查询

**触发词**：科研、实验室、论文、研究、科研经费

**数据源**：`universities.json` 中的 `科研概况` 字段 → 若不足，线上搜索补充

**线上搜索策略**：搜索 "{高校名} 科研经费 年报" 或 "{高校名} 国家重点实验室"

**输出模板**：
```
🔬 {高校名称} - 科研水平

• 国家重点实验室：{数量及名称}
• 科研经费：{年度数据}
• 代表性成果：{高影响力论文/项目}
• 优势研究领域：{基于A+学科推断}

⚠️ 注：科研数据来源于{来源}，时效为{年份}
```

### 4. 就业质量查询

**触发词**：就业、毕业生、薪资、工资、去向、就业率、校招

**数据源**：`universities.json` 中的 `就业概况` 字段 → 若不足，线上搜索 "{高校名} 就业质量报告 {最近年份}"

**输出模板**：
```
💼 {高校名称} - 就业质量

• 就业率：{数据}
• 主要去向行业：{行业列表}
• 代表性雇主：{企业名}
• 深造率：{国内考研+出国留学比例}
• 薪资水平：{数据，标注年份}

⚠️ 注：数据来源于{高校名}20XX年就业质量报告
```

### 5. 生源结构查询

**触发词**：多少学生、本硕博比例、男女比例、招生

**数据源**：`universities.json` 中的 `生源概况` 字段 → 线上搜索补充

**输出模板**：
```
👥 {高校名称} - 生源结构

• 在校生规模：{总数}
• 学历层次：本科 {X}%，硕士 {Y}%，博士 {Z}%
• 男女比例：{数据}
• 国际学生：{数据}
```

### 6. 院校筛选查询

**触发词**：有哪些、哪些学校、筛选、列表

**数据源**：`config.yaml`（按层次/地区/类型筛选）

**处理逻辑**：
1. 解析筛选条件：层次（985/211/双一流）、地区（省份/大区）、类型（理工/综合/师范等）
2. 从 config.yaml 中交叉筛选
3. 按层次优先级排序（985 > 211 > 双一流 > 其他）

**输出模板**：
```
🏛️ {筛选条件描述}（共{N}所）

1. {高校名} - {类型} - {城市}
2. {高校名} - {类型} - {城市}
...

💡 如需了解某一所高校的详细信息，请输入高校名称。
```

### 7. 高校对比查询

**触发词**：对比、比较、VS、和...哪个好

**处理逻辑**：
1. 识别要对比的2-3所高校
2. 分别从本地数据获取各校信息
3. 按维度横向对比输出

**输出模板**：
```
⚖️ {高校A} vs {高校B}

| 维度 | {高校A} | {高校B} |
|------|---------|---------|
| 院校层次 | ... | ... |
| 所在城市 | ... | ... |
| 优势学科 | ... | ... |
| A+学科数 | ... | ... |
| 就业去向 | ... | ... |
```

### 8. 综合画像查询

**触发词**：怎么样、介绍一下、了解

**处理逻辑**：聚合所有维度数据，给出全面画像

## 线上搜索策略

当本地数据不足时，按以下策略搜索：

| 查询类型 | 搜索关键词模板 | 优先目标网站 |
|----------|---------------|-------------|
| 就业数据 | "{高校} 就业质量报告 {当前年份-1}" | 各校就业信息网 |
| 科研数据 | "{高校} 科研经费 OR 重点实验室" | 各校官网/科研处 |
| 招生数据 | "{高校} 招生简章 {当前年份}" | 阳光高考平台 |
| 排名数据 | "{高校} 排名 软科 OR 校友会 {当前年份}" | 软科/校友会官网 |
| 综合介绍 | "{高校} site:教育部 OR site:各校官网" | 教育部/官网 |

**搜索注意事项**：
- 优先使用权威数据源（教育部、各校官网）
- 明确标注数据年份
- 避免使用非官方排名作为绝对依据
- 多个来源数据有冲突时，优先信任官方来源

## 输出规范

### 格式要求
- 使用 emoji 前缀区分信息维度（📚基础 📖学科 🔬科研 💼就业 👥生源）
- 关键数据加粗或用列表清晰展示
- 对比类查询用表格展示
- 每次输出末尾提示可追问方向

### 数据标注
- 学科评估数据标注"第四轮学科评估（2017年公布）"
- 就业/科研等时效性数据标注具体年份
- 线上搜索补充的数据标注来源 URL

### 不确定性处理
- 本地数据中找不到 → 明确告知"本地数据未收录该校，正在搜索补充..."
- 线上也找不到 → 告知"暂未找到该信息，建议访问 {官网URL} 查询"
- 数据可能过时 → 标注"该数据为{年份}数据，最新情况请参考官方发布"

## 约束

### 边界限制
- 本 Skill 专注于高校信息检索，不提供志愿填报建议或录取分数预测
- 不对高校做主观排名或"好坏"评价
- 薪资、就业等数据来源于各校公开报告，仅供参考
- 数据不作为正式决策依据

### 隐私保护
- 不收录个人隐私信息
- 不提供可定位个人的学生/教师信息
- 就业数据仅限群体统计层面

### 响应优先级
1. 准确性 > 完整性 > 速度（宁可少说不确定的，不要编造数据）
2. 本地缓存 > 线上搜索（确保基本响应速度）
3. 官方数据 > 第三方数据（确保数据可信度）
