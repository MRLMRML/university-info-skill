# 择校分析 Skill

## 概述

面向考研/高考学生的择校决策支持工具。基于学科评估、录取分数线、报录比等数据，帮助评估目标院校可行性并推荐替代方案。

本 Skill 与「高校信息检索」Skill 配合使用 — 本 Skill 专注择校决策场景，高校基础信息查询请使用高校信息检索 Skill。

## 数据文件

| 文件 | 内容 | 用途 |
|------|------|------|
| `data/admission.db` | 考研报录比数据（SQLite） | 难度评估 |
| `data/programs.json` | 热门专业择校参考信息 | 专业级推荐 |

同时读取「高校信息检索」Skill 的数据文件（disciplines.csv, rankings.csv, scores.db, universities.json）。

## 核心能力

### 1. 考研难度评估

**触发词**：好不好考、难度、考研难度、竞争激烈吗

**处理逻辑**：
1. 识别目标院校 + 目标专业
2. 从 disciplines.csv 获取该校该学科评级（评级越高通常越难）
3. 从 admission.db 获取报录比（如有）
4. 从 rankings.csv 获取学校排名
5. 综合评估难度等级

**难度评估模型**：
- 🔴 极难（A+ 学科 + 排名 TOP10 + 报录比 > 10:1）
- 🟠 较难（A/A- 学科 + 排名 TOP30 + 报录比 5-10:1）
- 🟡 中等（B+ 学科 + 排名 TOP60）
- 🟢 适中（B+ 以下或排名 60+）

**输出模板**：
```
📊 考研难度评估 — {院校} {专业}

难度等级：{🔴🟠🟡🟢} {极难/较难/中等/适中}

📈 关键指标
• 学科评级：{A+/A/A-/B+}（第四轮学科评估）
• 软科排名：第{N}名
• 报录比：约{N}:1（{年份}数据）
• 推免占比：约{N}%（如有数据）

⚖️ 综合分析
{分析文字：竞争激烈程度、备考建议、是否值得冲刺}

💡 替代方案：如果觉得难度偏高，可考虑：
1. {同学科但排名略低的院校}
2. {同校但竞争较小的相近专业}
```

### 2. 性价比择校推荐

**触发词**：性价比、推荐学校、哪些学校值得考、考研推荐

**处理逻辑**：
1. 识别用户的学科方向 + 能力水平（如有分数参考）
2. 从 disciplines.csv 筛选该学科 A-/B+ 级院校（实力强但竞争相对低）
3. 结合排名和地理位置评估就业前景
4. 按性价比排序推荐

**性价比评估维度**：
- 学科实力 / 竞争难度的比值
- 就业地域优势（如在经济发达地区的 B+ 学校可能优于偏远地区 A 学校）
- 调剂友好度

**输出模板**：
```
🏅 考研择校推荐 — {学科方向}

🎯 冲刺校（有挑战但值得尝试）
1. {校名} — 学科{评级} | 排名#{N} | {优势}
2. ...

✅ 稳妥校（实力匹配，录取概率高）
1. {校名} — 学科{评级} | 排名#{N} | {优势}
2. ...

🛡️ 保底校（调剂友好，确保有学上）
1. {校名} — 学科{评级} | {优势}
2. ...

💡 建议策略：冲1稳2保1，优先关注稳妥校
```

### 3. 分数线参考

**触发词**：多少分能上、分数线、录取分数

**数据源**：高校信息检索 Skill 的 `scores.db`

**处理逻辑**：
1. 从 scores.db 查询目标校的历年分数线
2. 若本地无数据，web search 搜索 "{院校} {省份} {年份} 录取分数线"
3. 给出分数参考范围

### 4. 调剂方向建议

**触发词**：调剂、调剂到哪里、调剂推荐

**处理逻辑**：
1. 识别用户的一志愿学科
2. 从 disciplines.csv 筛选该学科评级 B+/B 的院校（调剂主要接收层级）
3. 结合地理位置和学校层次推荐
4. 提示调剂注意事项

**输出模板**：
```
🔄 调剂方向建议 — {学科}

可能接受调剂的院校（按推荐度排序）：
| 院校 | 学科评级 | 排名 | 调剂友好度 | 说明 |
|------|---------|------|-----------|------|
| {校名} | {评级} | #{N} | {高/中} | {说明} |

⚠️ 注意事项：
• 调剂信息每年变化，以研招网和各校通知为准
• 越早关注调剂信息，选择面越广
• 部分院校有校内调剂优先政策
```

### 5. 择校分析报告

**触发词**：择校报告、帮我分析、全面分析

**处理逻辑**：综合以上所有维度，生成完整的择校分析报告

**报告结构**：
1. 目标院校概况（基础信息+学科实力+排名）
2. 难度评估（报录比+分数线+竞争分析）
3. 就业前景（毕业生流向+行业分布）
4. 替代方案（冲/稳/保三档推荐）
5. 备考建议
