# 输出模板

查询结果按以下模板格式化输出。使用 emoji 前缀区分维度，关键数据加粗，末尾提示可追问方向。

## 可视化规则

- **对比类查询**：使用 Markdown 表格 + 文字雷达图
- **排名/趋势类**：使用 Mermaid 柱状图（如环境支持）
- **流向分析**：使用文字桑基图/占比条
- **占比数据**：使用进度条可视化 `████████░░ 80%`

### 占比条（用于流向/深造率等）

```
█████████░ 90%  就业率
████████░░ 80%  深造率
██████░░░░ 60%  留本地
```

### 雷达图文字版（用于多维对比）

```
         学科 ★★★★★
         排名 ★★★★☆
         就业 ★★★★☆
         科研 ★★★☆☆
         师资 ★★★★☆
         生源 ★★★★☆
```
