# 高校信息检索 Skill

中国高校信息查询与情报分析工具，面向 AI Coding Agent（如 [OpenCode](https://github.com/opencode-ai)）。

覆盖全国 147 所双一流高校，支持 7 种查询类型，数据源自教育部官方文件。

## 能力概览

| 查询类型 | 示例 | 数据源 |
|----------|------|--------|
| 基础信息 | "北京大学的基本信息" | universities.json |
| 学科实力 | "清华有哪些A+学科？" | disciplines.csv |
| 科研水平 | "浙大有哪些国家重点实验室？" | universities.json + Web Search |
| 就业质量 | "北航毕业生去向" | universities.json + Web Search |
| 生源结构 | "复旦本硕博比例" | universities.json |
| 院校筛选 | "北京有哪些985高校？" | config.yaml |
| 高校对比 | "浙大和上交对比" | 多文件聚合 |

## 数据文件

```
data/
├── universities.json    # ~100所高校：层次/城市/类型/一流学科/就业/科研/生源
├── disciplines.csv      # 教育部第四轮学科评估 A 类学科（2017年公布）
└── config.yaml          # 985(39所)/211(115所)/双一流(147所)完整名单
```

### 数据可信度

| 数据类型 | 可信度 | 来源 |
|----------|--------|------|
| 985/211/双一流名单 | 权威 | 教育部公开文件 |
| 学科评估结果 | 权威 | 教育部第四轮学科评估原始数据 |
| 高校层次/城市/类型 | 权威 | 教育部/阳光高考平台 |
| 就业/科研/生源概况 | 近似参考 | 各校公开年报概括 |

## 安装使用

将本仓库克隆到 OpenCode 的 skills 目录：

```bash
git clone https://github.com/MRLMRML/university-info-skill.git ~/.opencode/skills/高校信息检索
```

安装后，当你向 AI Agent 提问中国高校相关问题时，Skill 会自动触发。

### 触发示例

```
> 北大的就业情况怎么样？
> 计算机学科哪些学校是A+？
> 上海有哪些985高校？
> 哈工大和北航对比一下
> 学科信息:浙江大学|计算机
```

## 约束与免责

- 本 Skill 专注信息检索，不提供志愿填报建议或录取预测
- 不对高校做主观好坏评价
- 就业/科研数据为概括性参考，具体以各校官方发布为准
- 不收录任何个人隐私信息
