# 输出模板

查询结果按以下模板格式化输出。使用 emoji 前缀区分维度，关键数据加粗，末尾提示可追问方向。

## 基础信息

```
📚 {高校名称}

🏛️ 基础信息
• 院校层次：{985|211|双一流}
• 办学性质：{公立/民办}
• 所在城市：{城市}
• 高校类型：{综合/理工/师范/...}
• 官方网站：{网址}
```

## 学科实力

```
📖 {高校名称} - 学科实力

🏆 A+学科（{N}个）
• {学科名} - 学科代码 {代码}

🥇 A学科（{N}个）
• {学科名}

🥈 A-学科（{N}个）
• {学科名}

🌟 双一流建设学科
• {一流学科列表}

📊 数据来源：教育部第四轮学科评估（2017年公布）
```

## 科研水平

```
🔬 {高校名称} - 科研水平

• 国家重点实验室：{数量及名称}
• 科研经费：{年度数据}
• 代表性成果：{高影响力论文/项目}
• 优势研究领域：{基于A+学科推断}

⚠️ 注：科研数据来源于{来源}，时效为{年份}
```

## 就业质量

```
💼 {高校名称} - 就业质量

• 就业率：{数据}
• 主要去向行业：{行业列表}
• 代表性雇主：{企业名}
• 深造率：{国内考研+出国留学比例}
• 薪资水平：{数据，标注年份}

⚠️ 注：数据来源于{高校名}20XX年就业质量报告
```

## 生源结构

```
👥 {高校名称} - 生源结构

• 在校生规模：{总数}
• 学历层次：本科 {X}%，硕士 {Y}%，博士 {Z}%
• 男女比例：{数据}
• 国际学生：{数据}
```

## 院校筛选

```
🏛️ {筛选条件描述}（共{N}所）

1. {高校名} - {类型} - {城市}
2. {高校名} - {类型} - {城市}
...

💡 如需了解某一所高校的详细信息，请输入高校名称。
```

## 高校对比

```
⚖️ {高校A} vs {高校B}

| 维度 | {高校A} | {高校B} |
|------|---------|---------|
| 院校层次 | ... | ... |
| 所在城市 | ... | ... |
| 软科排名 | ... | ... |
| A+学科 | ... | ... |
| 就业去向 | ... | ... |
| 师资（院士） | ... | ... |
```

## 录取分数线

```
📈 {高校名称} - 录取分数线

{年份}年各省录取最低分：
| 省份 | 科类 | 最低分 | 最低位次 |
|------|------|--------|----------|
| {省} | {科类} | {分} | {位次} |

⚠️ 注：分数线数据为概括性参考，实际以各省教育考试院公布为准
```

## 大学排名

```
🏅 {高校名称} - 大学排名

• 软科2024排名：第{N}名
• 总分：{分数}
• 省份：{省}
• 类型：{类型}

📊 数据来源：软科中国大学排名2024
```

## 师资实力

```
👨‍🏫 {高校名称} - 师资实力

• 两院院士：{N}人
• 国家杰青：{N}人
• 长江学者：{N}人

📊 数据来源：各校官网公开数据（{年份}）
```

## 智能推荐

```
🎯 智能推荐 - {需求描述}

推荐目标院校（按综合匹配度排序）：
| # | 院校 | 相关学科评级 | 软科排名 | 匹配理由 |
|---|------|-------------|---------|----------|
| 1 | {校名} | {A+/A/...} | #{N} | {理由} |

💡 以上推荐基于学科实力、大学排名、地区匹配综合评估
```

## 学科交叉

```
🔀 学科交叉查询 - {学科A} × {学科B}

同时在两个学科均表现优秀的院校：
| 院校 | {学科A} | {学科B} | 综合 |
|------|---------|---------|------|
| {校名} | {评级} | {评级} | {说明} |
```
