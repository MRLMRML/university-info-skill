# 线上搜索策略

本 Skill 采用「本地缓存 + 按需实时补充」架构。本地数据覆盖 39 所 985 核心字段，其余数据在用户查询时自动通过 web search 实时获取。这保证了数据的时效性和覆盖面。

## 搜索关键词模板

| 查询类型 | 搜索关键词 | 优先目标网站 |
|----------|-----------|-------------|
| 就业数据 | "{高校} 就业质量报告 {当前年份-1}" | 各校就业信息网 |
| 科研数据 | "{高校} 科研经费 OR 重点实验室" | 各校官网/科研处 |
| 招生数据 | "{高校} 招生简章 {当前年份}" | 阳光高考平台 |
| 排名数据 | "{高校} 排名 软科 OR 校友会 {当前年份}" | 软科/校友会官网 |
| 分数线 | "{高校} {省份} {年份} 录取分数线" | 各省教育考试院 |
| 综合介绍 | "{高校} site:教育部 OR site:各校官网" | 教育部/官网 |
| 师资数据 | "{高校} 院士 杰青 师资队伍" | 各校人事处/官网 |
| 实验室 | "{高校} 国家重点实验室 列表" | 各校科研处 |

## 注意事项

- 优先使用权威数据源（教育部、各校官网）
- 明确标注数据年份
- 避免使用非官方排名作为绝对依据
- 多个来源数据有冲突时，优先信任官方来源
- 搜索结果需要整合到统一输出格式中
- 线上搜索补充的数据必须标注来源 URL

## 按需实时补充场景

以下场景本地数据不完整，**必须自动触发搜索**：

| 场景 | 搜索关键词 |
|------|-----------|
| 非985就业数据 | "{高校} 毕业生就业质量年度报告" |
| 就业精确数字 | "{高校} 就业质量报告 {当前年份-1} 就业率 深造率" |
| 考研报录比 | "{高校} {专业} 报录比 {年份} 研究生招生" |
| 非985生源 | "{高校} 在校生规模 本科 研究生" |
| 师资精确 | "{高校} 两院院士 国家杰青 师资 site:edu.cn" |
| 缺失省份分数线 | "{高校} {省份} {年份} 录取分数线 最低分 位次" |

## 不确定性处理

- 本地数据不足 → 自动搜索补充，不需要告知用户"正在搜索"
- 搜到精确值 → 直接使用精确值，标注来源URL
- 搜不到 → 使用本地估算值，标注"为概括性参考，具体请查阅官方发布"
- 数据可能过时 → 标注"该数据为{年份}数据，最新情况请参考官方发布"
