# 线上搜索策略

当本地数据不足时，按以下策略使用 web search 补充。

## 搜索关键词模板

| 查询类型 | 搜索关键词 | 优先目标网站 |
|----------|-----------|-------------|
| 就业数据 | "{高校} 就业质量报告 {当前年份-1}" | 各校就业信息网 |
| 科研数据 | "{高校} 科研经费 OR 重点实验室" | 各校官网/科研处 |
| 招生数据 | "{高校} 招生简章 {当前年份}" | 阳光高考平台 |
| 排名数据 | "{高校} 排名 软科 OR 校友会 {当前年份}" | 软科/校友会官网 |
| 分数线 | "{高校} {省份} {年份} 录取分数线" | 各省教育考试院 |
| 综合介绍 | "{高校} site:教育部 OR site:各校官网" | 教育部/官网 |
| 师资数据 | "{高校} 院士 杰青 师资队伍" | 各校人事处/官网 |
| 实验室 | "{高校} 国家重点实验室 列表" | 各校科研处 |

## 注意事项

- 优先使用权威数据源（教育部、各校官网）
- 明确标注数据年份
- 避免使用非官方排名作为绝对依据
- 多个来源数据有冲突时，优先信任官方来源
- 搜索结果需要整合到统一输出格式中
- 线上搜索补充的数据必须标注来源 URL

## 不确定性处理

- 本地未收录 → "本地数据未收录该校，正在搜索补充..."
- 线上未找到 → "暂未找到该信息，建议访问 {官网URL} 查询"
- 数据可能过时 → "该数据为{年份}数据，最新情况请参考官方发布"
